# Plantel - Gestão de Animais

Cadastro e acompanhamento de animais com fichas detalhadas, controle de vacinação e árvore genealógica. Roda direto no navegador, sem instalação ou dependências.

<br />

</div>


## Sobre

**Plantel** é uma aplicação web leve para gestão de animais domésticos, de criação ou de trabalho. Desenvolvida em HTML, CSS e JavaScript puros, sem frameworks, sem build e sem servidor, basta abrir o `index.html` no navegador.

Pensada para criadores, clínicas veterinárias, fazendeiros e tutores que precisam de uma ferramenta simples e funcional para organizar seu plantel.

## Como Usar

```bash
git clone https://github.com/seu-usuario/plantel.git
cd plantel
open index.html
```

## Funcionalidades

- Cadastro com espécie, raça, sexo, peso, microchip e foto
- Status do animal (Ativo / Em tratamento / Inativo)
- Controle de vacinação com alertas automáticos
- Árvore genealógica visual
- Busca e filtro por nome, raça e espécie
- Tema claro e escuro
